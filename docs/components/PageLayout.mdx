# PageLayout Component

Provides a standard page structure with a fixed header and a main content area.

## Description

The `PageLayout` component renders a `<div>` container that uses flexbox to ensure a minimum height of the screen. It includes:

*   **Header:** A sticky `<header>` element with a light blue, semi-transparent background (`bg-blue-200/50`) and a bottom border. Its content spans the full width and has 24px horizontal padding (`px-6`). It contains:
    *   "My Site" text aligned to the left padding edge.
    *   An optional "Open Layer" (+) button aligned to the right padding edge.
*   **Main Content Area:** A `<main>` element for the primary page content. It uses the `container` class for responsive centering and width, plus explicit `px-6` padding for a guaranteed 24px horizontal margin.

## Props

| Prop                       | Type           | Required | Default | Description                                                          |
| :------------------------- | :------------- | :------- | :------ | :------------------------------------------------------------------- |
| `children`                 | `React.ReactNode` | Yes      | -       | The content to be rendered inside the main area.                       |
| `onOpenFloatingLayerClick` | `() => void`   | No       | -       | Callback function executed when the "Open Layer" (+) button is clicked. If not provided, the button is hidden. |

## Usage Example

```tsx
import PageLayout from '@/components/PageLayout/PageLayout';

export default function MyPage() {
  const handleOpenLayer = () => {
    console.log('Open layer button clicked!');
    // Typically, you would set state here to open the FloatingLayer
  };

  return (
    <PageLayout onOpenFloatingLayerClick={handleOpenLayer}>
      <h1>Page Title</h1>
      <p>This content will appear below the fixed header.</p>
    </PageLayout>
  );
}
```
